# Relaywright Notice

Relaywright is a free, self-hosted SMTP relay gateway distributed under the MIT License.

Release packages are self-contained .NET builds for Windows x64 and Linux x64. They include third-party open-source dependencies from the .NET, ASP.NET Core, EF Core, MailKit, SmtpServer, and SQLite ecosystems under their respective licenses.

Do not publish runtime `App_Data` directories, Data Protection keys, backup bundles, protected secret blobs, logs, or message spool files.
