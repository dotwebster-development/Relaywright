# Relaywright Brand Template

This template captures the reusable colors and assets from the first Relaywright logo concept.

## Assets

- Generated concept: `src/Relaywright.Web/wwwroot/brand/relaywright-logo-concept.png`
- Reusable SVG logo: `src/Relaywright.Web/wwwroot/brand/relaywright-logo.svg`
- Reusable SVG mark: `src/Relaywright.Web/wwwroot/brand/relaywright-mark.svg`
- Web favicon: `src/Relaywright.Web/wwwroot/favicon.svg`

## Palette

Use these tokens from `site.css` rather than hard-coded colors in new UI work:

| Token | Hex | Use |
| --- | --- | --- |
| `--brand-graphite` | `#283038` | Primary text, wordmark, strong UI surfaces |
| `--brand-graphite-strong` | `#182028` | Hover/pressed text and dark accents |
| `--brand-teal` | `#088090` | Primary action and active navigation color |
| `--brand-teal-strong` | `#007888` | Button hover and focused accent |
| `--brand-teal-soft` | `#e6f6f8` | Selected rows, active tabs, quiet highlights |
| `--brand-amber` | `#e8a820` | Queue/warning accent and small attention markers |
| `--brand-amber-soft` | `#fff6df` | Warning backgrounds |

## UI Usage

- Use teal for primary actions, active nav, focus outlines, and links.
- Use amber sparingly for queue/warning signals and small markers.
- Keep graphite as the main wordmark/text anchor.
- Keep page surfaces neutral so operational data remains easy to scan.
- Prefer the SVG mark for UI chrome and the generated PNG only as a visual concept reference.
